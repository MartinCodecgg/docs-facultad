# Resumen

Este texto consiste en la transcripción de una **entrevista y sesión de trabajo** orientada al desarrollo de un **sistema de seguimiento de trayectorias estudiantiles** para el departamento de ingeniería industrial. El cliente detalla que el objetivo principal es mitigar la **deserción temprana y tardía**, así como el alargamiento de las carreras, mediante la detección de **situaciones críticas** y la asignación de tutorías. Debido a la imposibilidad de usar bases de datos cerradas como el SIU Guaraní, se propone un sistema de **relevamiento voluntario** vinculado al campus virtual (Moodle) que capture datos cuantitativos y cualitativos. El sistema debe permitir la **parametrización de indicadores**, la gestión de alertas y la visualización de tableros de control tanto para la gestión académica como para el propio alumno. Finalmente, se establecen las pautas de trabajo bajo **metodologías ágiles**, enfatizando la importancia de la validación de datos, la protección de información sensible y la creación de prototipos funcionales.
